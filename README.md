# IP Subnet Calculator & Practice

Live site: **https://wattox00.github.io/ip**

## How to use the app
- **Switch sections**: Click the **"Switch to IP Calculator"** button to toggle between the Calculator and Practice sections.
- **IP Calculator**:
  - Enter an address in CIDR format, e.g. `192.168.100.1/24` in the input field and press Enter (or the UI will calculate automatically).
  - The results are shown in the output area and include network address, broadcast, usable hosts, etc.
- **Practice area**:
  - Click **Generate** to get a random IP to practice on.
  - Use the class toggles (Class A / B / C / Under) to restrict which class of IPs will be generated.
  - Fill in the fields for **Network**, **Broadcast**, **First IP**, and **Last IP** (both decimal and binary where requested).
  - Click **Check Result** to verify your answers.
- **Footer**: Click the grey footer box to hide/show it; the small arrow toggles visibility as well.
